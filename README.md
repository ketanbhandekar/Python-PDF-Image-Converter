PDF ↔️ IMAGE CONVERTER

This project provides simple utilities to:

.  Convert PDF files to images
.  Convert images to a PDF file

FEATURES:

.  Convert each page of a PDF into high-quality image(s)
.  Combine one or more images into a single PDF
.  Supports popular image formats (JPG, PNG, etc.)
.  Easy-to-use interface (CLI or GUI depending on your implementation)
